# Mouser Chief Collection

Get chief mouser data as JSON.

## Sources

Wikipedia: https://en.wikipedia.org/wiki/Chief_Mouser_to_the_Cabinet_Office

## Commands

### Install

```
 $ pipenv install
```

### Dump HTML

```
 $ pipenv run dump
```

### Extract values from HTML

```
 $ pipenv run extract
```

### Parse values

```
 $ pipenv run parse
```

### Format json file

```
 $ pipenv run format
```

### Lint

```
 $ pipenv run lint
```
